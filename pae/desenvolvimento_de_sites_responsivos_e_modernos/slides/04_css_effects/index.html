Transitions

Transições de propriedades de maneira suave

Propriedades de Transição CSS
transition-property: qual propriedade CSS irá transicionar transition-duration: quanto tempo a transição irá durar transition-timing-function: qual a curva de velocidade da transição transition-delay: quanto tempo a transição irá esperar para começar transition: Atalho para utilizar as 4 propriedades em uma só
transition-timing-function
ease - efeito de transição com um início lento, depois rápido e termina lento linear - efeito de transição com a mesma velocidade do início ao fim ease-in - efeito de transição com um início lento ease-out - efeito de transição com um final lento ease-in-out - efeito de transição com início e fim lentos 
            
              /* com transição */
              div {
                width: 100px;
                transition: width 600ms linear 200ms;
              }
              div:hover {
                width: 200px;
              }

              /* sem transição */
              div {
                width: 100px;
              }
              div:hover {
                width: 200px;
              }
