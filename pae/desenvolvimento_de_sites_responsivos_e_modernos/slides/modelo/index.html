Transform 2D
  • translate() -> Move o elemento da posição atual (top, left)
  • rotate() -> Rotaciona o objeto
  • scale() -> Mexe na escala do objeto
  • skewX() -> Move o objeto no eixo X da tela
  • skewY() -> Move o objeto no eixo Y da tela
  • skew(X, Y) -> Move o objeto no eixo X e no Y da tela
  • matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY()) -> Combina todos os métodos 2D juntos
Bower

É um gerenciador de pacotes. (bower.io)

Como?

npm install -g bower

bower init (Isso vai gerar um arquivo chamdo bower.json)

bower install --save NOME_DO_PACOTE

Bower init
-g, --save, --save-dev ???

-g -> Global, acessível em qualquer ponto de sua máquina (para todos os usuário que tenham acesso a pasta global do node_modules)

--save -> Dependência que vai junto com seu projeto para produção, sem ela a aplicação "quebra" (AngularJS)

--save-dev -> Dependência apenas para auxiliar/melhorar o desenvolvimento. (Jasmine)

Observações
version: Versão exata do pacote
>version: Maior do que a versão, >=version, menor, menor que igual
~version: Aproximadamente /// ^version: Compatível com 'version'
1.2.x: 1.2.0, 1.2.1, etc., mas não 1.3.0
http://...: Passar uma URL diretamente
* ou "": qualquer versão
version1 - version2: mairo que ou igual a version1 e menor que ou igual a version2
range1 || range2: um ou outro
git...: repositório git (URL)
user/repo: repositório git (nome_do_usuario/nome_do_repositorio)
tag: latest, beta (etc)
local path: caminho de um modulo local seu
